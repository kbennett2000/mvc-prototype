# Code of Conduct

This project welcomes contributors of every background. To keep things constructive:

- **Be patient.** Many contributors are church volunteers learning as they go.
- **Be specific.** "This doesn't work" is harder to act on than "I clicked X and got error Y."
- **Be kind.** Critique code, not people.
- **Avoid theological disagreements in technical discussions.** This template is used by churches of many traditions. Keep contribution discussions focused on the software.

Unacceptable: harassment, personal attacks, sustained disruption.

Report concerns to the maintainers via GitHub issue or, for sensitive matters, email a maintainer privately (contact info in the project README).

We follow the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) for anything not spelled out above.
