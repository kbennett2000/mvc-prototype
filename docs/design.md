# Design — Majestic View Church Prototype

Warm, grounded, mountain-state Christian community. Avoid corporate blue/indigo, avoid pastel-cute. Reference targets: Redeemer NYC, The Village Church — restrained, editorial, sophisticated.

## Palette options

### Option A — Sandstone & Sage (chosen)
Warm cream foundation with a deep sage primary and a terracotta accent reserved for CTAs and emphasis. Reads as welcoming and confident; the sage keeps it from feeling like a generic "warm beige" template.

| Token              | HSL                | Hex     | Use                              |
| ------------------ | ------------------ | ------- | -------------------------------- |
| background         | `38 30% 96%`       | #F8F4EC | Page background                  |
| foreground         | `30 12% 15%`       | #2A2520 | Body text                        |
| card               | `0 0% 100%`        | #FFFFFF | Card surfaces                    |
| primary (sage)     | `120 14% 32%`      | #475C49 | Primary buttons, brand accents   |
| accent (terracotta)| `15 55% 47%`       | #BB5A37 | Hero CTA, link hover, highlights |
| muted              | `36 24% 90%`       | #ECE4D6 | Subtle backgrounds, dividers     |
| muted-foreground   | `30 10% 40%`       | #6B635A | Secondary text                   |
| border             | `35 20% 84%`       | #DDD3C1 | Hairlines                        |

### Option B — Mountain Morning
Soft alpine cream with deep evergreen primary and a brass/amber accent. Feels more "Colorado outdoors." Slightly cooler than A.

- background `#F4F2EC` · foreground `#1F2622` · primary `#2F4A3A` (evergreen) · accent `#B8842F` (brass)

### Option C — High Desert
Warmer still, with burnt sienna primary and olive accent. Most "high plains" feeling but risks looking rustic if not paired with restrained typography.

- background `#FBF6EE` · foreground `#2B221A` · primary `#A24A2A` (sienna) · accent `#7A7B3F` (olive)

## Decision
**Option A — Sandstone & Sage.** Sage as primary (rather than as accent) gives the brand more "personhood" than a brown-on-cream system, and terracotta as accent provides the warm punch a church homepage needs without crossing into Hallmark territory.

## Type
- Sans: **Inter** for UI/body — neutral, sober.
- Serif: **Fraunces** for display headings (hero, section titles) — adds editorial warmth without feeling vintage.

## Density
Generous vertical rhythm. Sections separated by 80–120px on desktop, 56–72px on mobile. Cards use 24–32px padding. Letting things breathe is a deliberate signal that we're not a startup landing page.
