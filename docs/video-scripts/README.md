# Video script templates

Short scripts for screencast walkthroughs that complement the written guides. Each script is structured so someone with a screen-recording tool (Loom, OBS, QuickTime) can hit Record and follow it.

Most scripts are 1.5–3 minutes. Editor-track videos use the same exact CMS clicks as the written guide they pair with.

## Script format

Each script has four sections:

1. **Open with** — short greeting + what the viewer will learn (5–10 sec)
2. **Steps to record** — the on-screen actions, with what to say
3. **Wrap** — confirmation + pointer to next step (10 sec)
4. **On-screen text / captions** — text overlays to add in post

Bold things to **say**. Bracketed `[things]` are **actions** (clicks, scrolls).

## Tips

- Record in 1080p with the browser zoomed to ~125% so text is readable on phones.
- Use a window size of 1280×720 — fits cleanly on YouTube without cropping.
- Don't say "um." Just stop and start the sentence over; cut in post.
- Mention the companion written doc at the end — different viewers prefer different formats.
