# Video: Edit a page

**Length:** 1.5 minutes
**Audience:** Editors updating the "Our Story" section or similar prose pages.
**Companion guide:** [03 Edit a page](../for-editors/03-edit-a-page.md)

## Open with (10 sec)
**Say:** "In this video I'll show you how to edit a page like 'Our Story.' It takes about a minute."

## Steps (1 min 10 sec)

### 1. Open Pages (10 sec)
[Click **Pages** in the left sidebar of `/admin/`.]
[Click **Our Story**.]

### 2. Use the WYSIWYG editor (40 sec)
**Say:** "The body field is a what-you-see-is-what-you-get editor — like Google Docs. Type your changes directly."
[Highlight a sentence, click the bold button.]
[Place cursor on a new line, click the heading button.]
[Show the link button — paste a URL.]

**Say:** "Use the toolbar for bold, headings, links. You don't write any code."

### 3. Save and publish (20 sec)
[Click **Save**, then **Status: Ready for Review**, then **Publish**.]
**Say:** "Save, mark as Ready for Review, click Publish. It'll appear on the About page after a volunteer approves."

## Wrap (10 sec)
**Say:** "And that's it. Same pattern works for any other page in the Pages collection."

## Captions
- 0:00 — "Editing a page in your church website"
- 0:30 — "WYSIWYG — write naturally, no code"
- 1:10 — "Save → Ready for Review → Publish"
