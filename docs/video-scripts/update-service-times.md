# Video: Update service times

**Length:** 1 minute
**Audience:** Editors changing the Sunday service time, office hours, or any church-wide setting.
**Companion guide:** [05 Update service times](../for-editors/05-update-service-times.md)

## Open with (10 sec)
**Say:** "In this video I'll show you how to update the Sunday service time. The same approach works for the address, phone, and any other church-wide setting. It takes about a minute."

## Steps (40 sec)

### 1. Open Site Settings (10 sec)
[Click **Site Settings** → **Church Info & Site Copy**.]

### 2. Edit the service time (20 sec)
[Scroll to the **Sunday Service** section.]
[Edit the **Service Start Time** field.]
**Say:** "Find Sunday Service. Change the start time to whatever the new time is. Format it like '10:00 AM' or '10:30 AM'."

### 3. Save and publish (10 sec)
[Click **Save** → **Status: Ready for Review** → **Publish**.]

## Wrap (10 sec)
**Say:** "Once a volunteer approves, the new time appears everywhere on the site — homepage, footer, Plan a Visit, the About page — all automatically. No editing each page separately."
